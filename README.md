## Tech Stack
- TypeScript
- Vite
- React
- Redux
- JSON Server
- Axios
- SCSS Modules


### Dependency installation
To install dependencies, run the command:
```sh
$ pnpm i
```

### Launching Development server
To start the server for development, run the command:
```sh
pnpm run dev
```

### Run serve
```sh
npx json-server db.json
```
